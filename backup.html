Reset Data

Tindakan reset akan menghapus seluruh isi tabel yang dipilih secara permanen. Pastikan Anda sudah melakukan backup CSV terlebih dahulu sebelum melanjutkan.
Restore Data
Unggah file CSV hasil backup untuk mengembalikan data ke tabel tujuan. Pastikan format kolom pada file CSV sesuai dengan struktur tabel.
